# Dashboard 页面设计

## 路由与权限

- 路由：`/dashboard`
- 权限：`ADMIN`

## 页面目标

- 展示平台运营总览与核心指标。

## 核心模块

- 核心 KPI 卡片
- 趋势图（请求、Token、成本、延迟）
- 模型或渠道分布
- 最近异常/事件摘要

## 接口契约

- `GET /api/admin/dashboard`

## 关键交互

- 时间维度切换触发整页刷新。
- 图表与表格口径统一。

## 状态与异常

- 首屏加载态。
- 无数据返回零值与空图，不报错。

## 验收标准

- 页面数据全部来自后端真实接口。
- 指标口径在前后端一致。
