# 冗余代码审计任务单

- 任务名称：FirstApi 项目冗余代码审计
- 任务类型：中任务 / 代码审阅
- 目标：识别 frontend / backend / docs / scripts 中的冗余、重复、未使用或明显过时的代码与资产
- 范围：
  - `frontend/`
  - `backend/`
  - 根目录审计文档与脚本
- 非目标：
  - 不直接删除代码
  - 不重构业务逻辑
  - 不强行判断当前未完成重构分支上的临时改动为“垃圾代码”
- 输出要求：
  1. 标出高置信度冗余项
  2. 标出低置信度、需人工确认项
  3. 给出文件路径和理由
  4. 区分前端 / 后端 / 文档脚本
